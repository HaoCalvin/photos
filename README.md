```markdown
Photopia — 去中心化照片分享（静态前端 + Gun.js）
说明文档

主要特性
- 响应式 UI：手机与桌面均适配
- 注册/登录（邮箱作为用户名）
- 照片上传，要求输入至少一个关键词
- 搜索关键词或用户名
- 隐私设置（私密照片仅上传者可见）
- 点赞、统计、热门排序（实时）
- 管理员删除图片
- 图片可放大查看
- 用户可编辑个人信息（头像、名称、简介）
- 三种主题：深色 / 浅色 / 白色
- 传输使用 HTTPS，私密图片在客户端额外加密

部署（建议）
1. 将 index.html、styles.css、app.js、README.md 上传到任意静态托管（GitHub Pages、Netlify、Cloudflare Pages 等）。
2. 在手机浏览器访问站点地址即可使用。首次打开可能需要一点时间与 Gun peers 同步。

安全与限制
- 本项目使用公开的 Gun relay peers（无需你自建服务器），因此服务端数据由公共节点转发与缓存。传输层使用 HTTPS。私密图片在客户端会额外用 Gun SEA 加密。
- 若你想完全控制数据与节点，请自行部署 Gun peer 服务器并在 app.js 中替换 peers 地址。
```
